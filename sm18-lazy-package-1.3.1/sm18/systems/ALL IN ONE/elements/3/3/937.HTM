Title: 章2 景观生态学的理论基础

景观生态学是一门交叉性、综合性和实用性很强的学科。在景观生态学形成和发展过程中，必然需要很多学科的理论和方法支撑。在此基础上，景观生态学通过兼容并蓄、集成创新形成了自身独特的学科架构。所以，了解不同学科理论对景观生态学的莫基作用，可以更好地开展景现生态学的理论和应用研究。以下将重点介绍对于现代景观生态学基础理论和基本概念框架发展起到积极推动作用的等级理论、地域分异理论、岛屿生物地理学理论、复合种群理论和渗透理论5种理论体系。

当然，对景观生态学的发展起推动作用的理论远不止这些，在实际研究和应用中，还需要结合更多其他学科的理论和方法，如景观格局分析需要借助于遥感科学和图论，生态过程分析需要借助于系统论和水文学，景观动态分析需要借助于时间序列分析和多种生态机理模型，生态服务研究需要借助于自然资源学和生态经济学，景观管理研究需要借助于生态评价、生态规划、生态恢复和生态管理等，需要根据工作需要对相关的理论与方法进行更详细的了解。
